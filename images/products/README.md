# Product Images

This folder contains product images for the Digital PC website.

## Structure

Each product has its own folder named after the product slug:
```
/products/
  /pk-i7-14700kf-rtx5070-32gb-ddr5/
    1.jpg
    2.jpg (optional)
    3.jpg (optional)
  /pk-rtx5070-r7-7700-ddr5-32gb/
    1.jpg
    ...
```

## How to Update Images

1. **Replace existing images**: Simply drop new `.jpg` files into the product folder
2. **Image naming**: Use `1.jpg`, `2.jpg`, `3.jpg` etc.
3. **Recommended size**: 800x800px minimum, square aspect ratio
4. **Format**: JPG or JPEG (PNG also supported but larger file size)

The website will automatically load images from these folders - no code changes needed!
