Digital PC (далі — «ми», «нас», «наш») поважає вашу приватність та зобов’язується захищати ваші персональні дані. Ця Політика конфіденційності пояснює, як ми збираємо, використовуємо, зберігаємо та захищаємо вашу інформацію під час використання сайту digitalpc.com.ua (далі — «Сайт»).

1. Які дані ми збираємо

Ми можемо збирати наступну інформацію:

  • Ім’я, прізвище, номер телефону, email — при оформленні замовлення або зв’язку з нами.
    • Дані про ваш пристрій (IP-адреса, тип браузера, операційна система) — для аналітики та покращення роботи Сайту.
      • Файли cookie — для збереження налаштувань та персоналізації.

        • Ми не збираємо платіжні дані — оплата здійснюється через сторонні сервіси (Prom.ua, OLX, банківські перекази).

        2. Як ми використовуємо ваші дані

        • Для обробки замовлень та доставки товарів.
          • Для зв’язку з вами (підтвердження замовлення, консультації).
            • Для покращення роботи Сайту та маркетингу (анонімна аналітика).
              • Для виконання вимог законодавства України.

                • 3. Захист даних

                Ми використовуємо сучасні технічні та організаційні заходи для захисту ваших даних від несанкціонованого доступу, втрати чи розголошення. Однак жоден спосіб передачі даних через Інтернет не є 100% безпечним.

                4. Передача даних третім особам

                Ми можемо передавати ваші дані:

                • Кур’єрським службам (Нова Пошта) для доставки.
                  • Платіжним системам та партнерам (Prom.ua, OLX).
                    • У випадках, передбачених законодавством України.

                      • Доступу до ваших персональних даних.

                              • Для реалізації прав звертайтеся за контактами нижче.
